# cemrecodes.github.io

My github.io portfolio website.

## How To Run

After installing go to the directory where the project is installed.

```bash
  docker build -t cemrecodes-web .
  docker run -p 8080:80 cemrecodes-web
```
After the command is completed you can see the project running on [https://localhost:8080](https://localhost:8080)
